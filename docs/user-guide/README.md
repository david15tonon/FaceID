# User Guide

## Getting Started

1. **Register**: Create an account
2. **Enroll Face**: Upload or capture your face
3. **Verify**: Verify your identity using face recognition

## Features

- Real-time face recognition
- Secure authentication
- Analytics and reporting
- Audit logs

## FAQ

**Q: How secure is face recognition?**
A: We use military-grade encryption and bcrypt hashing.

**Q: Can I enroll multiple faces?**
A: Yes, you can enroll multiple faces per account.
