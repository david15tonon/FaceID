# Getting Started with FaceID

## Overview

FaceID is a comprehensive face recognition system that allows you to enroll faces and verify identities with high accuracy.

## Quick Start

### 1. Sign Up

1. Navigate to the landing page
2. Click "Start Free" or "Sign up"
3. Fill in your details:
   - Full Name
   - Email Address
   - Password (min 8 characters)
4. Click "Create Account"

### 2. Login

1. Go to the login page
2. Enter your email and password
3. Click "Login"
4. (Optional) Complete MFA if enabled

### 3. Enroll Your Face

1. Navigate to "Enroll Face" from the dashboard
2. Choose enrollment method:
   - **Camera Capture**: Use your webcam for live capture
   - **Upload Image**: Upload an existing photo
3. Follow the instructions:
   - Position face in frame
   - Ensure good lighting
   - Remove glasses if possible
   - Look directly at camera
4. Click "Capture Face" or upload image
5. Confirm the captured image
6. Your face is now enrolled!

### 4. Verify Identity

1. Navigate to "Verify" from the dashboard
2. Choose verification method:
   - **Camera Capture**: Use webcam
   - **Upload Image**: Upload photo
3. Submit the image
4. View verification result:
   - Match found with confidence score
   - No match found

## Dashboard

The dashboard provides:
- Total number of recognitions
- Accuracy rate
- Number of enrolled faces
- Recent activity feed

## Best Practices

### For Enrollment
- Use high-quality, well-lit photos
- Face should be clearly visible
- Avoid sunglasses or masks
- Multiple angles can improve accuracy
- Re-enroll if accuracy is low

### For Verification
- Use similar lighting conditions as enrollment
- Face should be front-facing
- Maintain consistent distance from camera
- Avoid obstructions

## Troubleshooting

### Face Not Detected
- Ensure face is clearly visible
- Check lighting conditions
- Move closer to camera
- Remove obstructions

### Low Confidence Score
- Re-enroll with better quality image
- Ensure consistent lighting
- Update enrolled face if appearance changed

### Verification Failed
- Verify you're enrolled in the system
- Check image quality
- Ensure face is clearly visible
- Contact support if persistent
